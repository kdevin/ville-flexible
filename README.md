# Activation Service

[![python](https://img.shields.io/badge/Python-3.12-3776AB.svg?style=flat&logo=python&logoColor=white)](https://www.python.org)
[![fastapi](https://img.shields.io/pypi/v/fastapi?color=%252334D058&label=fastapi)](https://pypi.org/project/fastapi)

Activation Service is a FastAPI service handling TSO requests.

## Useful commands

### Create the virtualenv

    virtualenv activation-service

### Activate the virtualenv

    source activation-service/bin/activate

### Freeze your Python dependencies

    pip freeze > requirements.txt
